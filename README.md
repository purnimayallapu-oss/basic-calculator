# 🧮 Basic Calculator

This is my **first web development project**, built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations and helped me understand how real applications handle logic and edge cases.

---

## 🚀 Live Demo
🔗 [View Live Project](https://purnimayallapu-oss.github.io/basic-calculator/)

---

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

---

## ✨ Features
- Addition  
- Subtraction  
- Multiplication  
- Division  
- Handles division by zero with a user-friendly message ("Cannot divide by zero")
